파일명 일괄 수정기 사용법

파일을 업로드하지 않고 선택한 컴퓨터의 폴더 안에서 작업합니다.
  1. 폴더 선택「폴더 선택」을 누르고 이름을 바꿀 파일이 있는 폴더를 선택합니다.
  2. 새 이름 입력새 이름 칸을 직접 수정하거나 찾아 바꾸기, 앞·뒤 붙이기, 번호 만들기를 사용합니다.
  3. 엑셀에서 여러 이름 붙여넣기엑셀의 이름 한 열을 복사한 뒤 시작할 새 이름 칸을 클릭하고 붙여넣으면 아래 칸까지 차례로 입력됩니다.
  4. 변경 내용 확인현재 이름과 새 이름을 비교합니다. 「확장자 자동 유지」를 켜두면 .pdf 같은 기존 확장자가 유지됩니다.
  5. 전체 반영「전체 반영」을 눌러 실제 파일명을 바꿉니다. 브라우저가 수정을 차단하면 Windows 실행파일 다운로드를 선택합니다.
  6. 차선책 실행내려받은 「파일명_변경_실행.bat」를 대상 파일들이 있는 폴더로 옮긴 뒤 두 번 클릭합니다.
