📘 문항 배점 생성기 설명서

이 도구는 평가 문항의 선택형/서답형 점수와 상·중·하 난이도 비율을 입력하면, 조건에 맞는 배점표를 자동으로 만들어 주는 웹툴입니다. 선택형은 0.1점 단위, 서답형은 1점 단위로 생성됩니다.

⚡ 간편모드 사용 순서

  1. 전체 문항 수를 입력합니다.
  2. 총점과 선택형 비율 또는 선택형/서답형 점수를 입력합니다.
  3. 상·중·하 비율을 입력합니다. 합계는 100%여야 합니다.
  4. 배점 생성하기를 누르면 결과가 만들어집니다.

⚙️ 기본모드 사용 순서

  1. 선택형/서답형 문항 수를 직접 입력합니다.
  2. 상·중·하 문항 수와 점수합을 직접 조정합니다.
  3. 배점 생성을 눌러 조건에 맞는 배점을 만듭니다.
  4. 간편모드 결과를 불러온 뒤 세부 수정할 때 쓰면 좋습니다.

🔢 문항 수 기준 / 점수 합 기준

  • 문항 수 기준: 상·중·하 문항 개수 비율을 목표에 가깝게 맞춥니다.
  • 점수 합 기준: 상·중·하 난이도별 배점 합계를 목표 비율에 가깝게 맞춥니다.
  • 보통은 문항 수 기준으로 시작한 뒤, 필요하면 기본모드에서 조정하는 방식이 편합니다.

📋 결과 활용

  • 표 복사: 엑셀, 한셀, 구글 스프레드시트에 바로 붙여넣을 수 있습니다.
  • CSV 내려받기: 문항, 유형, 난이도, 배점이 들어간 파일을 저장합니다.
  • 기본모드로 수정: 자동 생성 결과를 세부 조정할 수 있습니다.

🚧 생성이 실패할 때

실패가 뜨는 경우는 대부분 조건이 너무 빡빡할 때입니다. 예를 들어 상 문항 수는 많은데 상 점수합이 너무 낮거나, 하 문항 수는 적은데 하 점수합이 너무 높은 경우입니다. 이럴 때는 상·중·하 비율을 조금 바꾸거나, 기본모드에서 점수합을 조정해 주세요.

💡 추천 사용 예시

  • 22문항, 100점 시험이라면 총점 100, 선택형 비율 70, 난이도 비율 상 20 / 중 50 / 하 30으로 시작해 보세요.
  • 결과가 마음에 들지 않으면 다시 생성을 눌러 다른 배점 조합을 만들 수 있습니다.
  • 최종 확인은 문항 순서 배정표에서 유형, 난이도, 배점을 같이 보면서 하면 좋습니다.
