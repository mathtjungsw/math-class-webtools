03 · 추세선 분석

데이터를 가장 잘 설명하는 직선 찾기

y = ax + b

2개 이상의 데이터를 준비하고 추세선을 만들어 보세요.

계산된 추세선 y = 0x + 0
기울기 a-
y절편 b-
데이터 수 n-
결정계수 R²-

결과 해석

  • 기울기는 x가 1 증가할 때 y가 얼마나 변하는지를 나타냅니다.
  • y절편은 x가 0일 때 예상되는 y값입니다.
  • R²는 직선이 데이터를 설명하는 정도이며 1에 가까울수록 설명력이 높습니다.
  • 추세선은 경향을 보여 줄 뿐, 항상 정확한 미래 값을 보장하지는 않습니다.
