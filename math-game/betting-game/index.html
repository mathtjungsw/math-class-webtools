먼저 Apps Script 코드를 준비하세요

아래 버튼으로 전체 코드를 복사한 뒤 Apps Script 편집기에 그대로 붙여넣으면 됩니다.

각각 2~20 사이에서 설정할 수 있습니다. 라운드가 12개라면 각 팀은 1~12를 한 번씩 사용합니다.

복사한 코드를 저장·웹 앱으로 배포한 뒤 생성된 /exec URL을 입력하세요. 게임 생성 시 ‘시트1’ 게임 영역이 새로 만들어집니다.

처음 사용하시나요?관리자 사용 설명서펼쳐보기
1

스프레드시트 준비

새 구글 스프레드시트를 만들고 시트 이름을 시트1로 둡니다. 표는 직접 만들지 않아도 됩니다.

2

Apps Script 붙여넣기

스프레드시트에서 확장 프로그램 → Apps Script를 열고, 이 도구 폴더의 Code.gs 전체를 붙여넣어 저장합니다.

3

웹앱 배포

배포 → 새 배포 → 웹 앱을 선택합니다. 실행 사용자는 ‘나’, 액세스 사용자는 수업 환경에 맞게 설정한 뒤 생성된 /exec URL을 복사합니다.

4

게임 생성

게임 생성은 이전 제출·점수·참여 코드를 모두 지우고 처음부터 시작합니다. 기존 진행 상태를 이어갈 때만 기존 게임 불러오기를 사용합니다.

5

라운드 운영

접속이 어려운 팀은 현재 라운드의 직접 입력을 눌러 대신 제출할 수 있습니다. 게임판 맨 아래의 점수 합계와 최종 순위를 확인합니다.

6

문제 해결

상태가 바뀌지 않으면 동기화를 누릅니다. 권한 오류는 웹앱 배포의 액세스 권한을, 연결 오류는 URL 끝이 /exec인지 확인합니다.
