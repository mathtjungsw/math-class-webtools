# 독점 보드게임

구글 시트와 Apps Script를 이용해 관리자 모드와 참여자 모드로 진행하는 가격 결정 전략 게임입니다.

## 진행 방식

- 관리자 모드에서 Apps Script 웹앱 URL, 구글 시트 주소, 팀 수를 입력해 게임을 생성합니다.
- 팀 수는 3팀부터 8팀까지 선택할 수 있습니다.
- 각 팀은 12개월 동안 매월 A, B, C 중 하나의 가격을 제출합니다.
- A는 높은 가격, B는 중간 가격, C는 낮은 가격입니다.
- 각 팀은 A를 최소 4번 사용해야 하고, C는 최대 4번까지만 사용할 수 있습니다.
- 관리자는 제출 현황만 본 뒤 전체 공개를 눌러 선택과 이익을 한 번에 공개합니다.
- 참여자는 게임 중 규칙과 현재 팀 수의 점수표를 언제든 확인할 수 있습니다.
- 최종 결과는 화면 인쇄 또는 CSV 저장으로 출력할 수 있습니다.

## 점수표

- 3팀 모드는 원본 엑셀 점수표를 그대로 사용합니다.
- 4~8팀 모드는 가격의 희소성, 높은 가격의 보상, 같은 가격 선택의 혼잡도를 반영해 자동 생성합니다.
- 생성된 점수표는 관리자 화면과 구글 시트의 점수표 영역에 함께 표시됩니다.

## 파일

- `index.html`: 관리자/참여자 화면
- `style.css`: 반응형 화면 스타일
- `app.js`: 화면 동작, JSONP 통신, 점수표 생성
- `Code.gs`: 구글 Apps Script 백엔드
